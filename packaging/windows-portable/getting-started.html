首次配置流程

  1. 1

    确认 Docker 已启动

    便携包需要 PostgreSQL 和 Redis。若 Start-Customer-AI.bat 能跑到本页，通常表示 Docker 已就绪。

  2. 2

    填写 LLM（Agnes）与 Embedding

    编辑便携包根目录 .env。默认直连 Agnes，无需启动本机 OpenClaw。至少配置：

    • LLM_PROVIDER=agnes（默认）
    • AGNES_API_URL、AGNES_API_KEY、AGNES_MODEL
    • EMBEDDING_API_KEY 等向量检索相关项

    若改用本机 OpenClaw，设置 LLM_PROVIDER=openclaw 后再启动网关。

  3. 3

    加载 Chrome 扩展

    打开 chrome://extensions，开启开发者模式，加载本便携包目录：

    extensions\customer-ai-rpa

  4. 4

    登录美团经营宝并开启插件

    在 Chrome 中人工登录客服页面。点击插件图标，确认显示「已连接本地服务」，并勾选：

    • 启用页面监听
    • 自动处理未读会话
    • 允许自动点击发送（测试通过后开启）
  5. 5

    上传门店知识库

    打开知识库管理页，上传门店地址、套餐、退款规则等资料。AI 回复必须基于知识库检索结果。

  6. 6

    配置 RPA 客户白名单

    在右侧「客户白名单」中填写可自动回复的客户 ID/昵称。美团灰度建议先加测试客户；全部清空后表示所有客户都可以回复。抖音默认也可留空。

客户白名单

每行一个客户 ID 或昵称，也可用英文逗号分隔。清空并保存后，对应平台对所有客户开放自动回复。

安全提示：便携包默认建议先在草稿模式验证。开启 RPA_AUTO_SEND_ENABLED=true 前，请确认知识库、白名单和发送按钮选择器均正确。 不要删除 data 目录，否则登录态会丢失；也不要把客户使用后的整个目录对外转发。
